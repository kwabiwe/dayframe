# Codex Feature Plans

Generated feature plans go here.

Recommended prompt:

```text
Read .codex/prompts/plan-feature.md and create a plan for: <feature or phase name>
```

Each plan should be complete enough that a fresh Codex thread can execute it with:

```text
Read .codex/prompts/execute-plan.md and execute .codex/plans/<plan-file>.md
```
